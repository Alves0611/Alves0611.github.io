Application Auto Scalable

Este projeto implementa uma arquitetura de alta disponibilidade na AWS usando o Terraform para provisionamento de recursos. A arquitetura consiste em instâncias EC2 em sub-redes privadas distribuídas em três Zonas de Disponibilidade, um Grupo de Auto Scaling para gerenciar essas instâncias, um Balanceador de Carga para distribuir tráfego e uma instância de banco de dados RDS em uma sub-rede privada. Uma instância adicional em uma sub-rede pública foi criada para simular um servidor Jenkins.

GitHub

AWS EKS Module

Este projeto Terraform permite a implantação automatizada de um cluster Amazon EKS (Elastic Kubernetes Service) na AWS, utilizando uma arquitetura modularizada. Os submódulos incluem a configuração de balanceamento de carga com o AWS Load Balancer Controller, a criação e gerenciamento do cluster EKS, grupos de nós gerenciados e a configuração da rede subjacente. Com esta solução, você pode provisionar um ambiente Kubernetes escalável, seguro e de alto desempenho, simplificando a administração e a manutenção da infraestrutura de contêineres na nuvem
